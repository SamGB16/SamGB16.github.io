Netlify-hosted demo that calculates server/bartender earnings — small web utility built with HTML/JS.

Case Study

Problem

Provide a simple calculator for servers/bartenders to estimate earnings for a shift with tips and hours.

Approach

Small static HTML/JS form that accepts hours, hourly rate, and tip total, then calculates earnings. Prioritized clear UX and deployability.

Outcome

Deployed site (Netlify) with a working demo. Demonstrates ability to ship a live tool recruiters can click and evaluate in seconds.

What I learned

How to quickly iterate a small web utility, deploy to Netlify, and write concise README and artifact notes for recruiters.
